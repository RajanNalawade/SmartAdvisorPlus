General

How can I return to Home screen from any activity?

Option to return back to ‘home’ is located in drop down menu on top-right corner of application.

How can I change application password?

To change password, user need to go to ‘Settings’ from drop down menu on top-right corner. User can select ‘change password’ option in ‘Setting’ to set new password.

How can I view my profile?

Option to view profile information is located in drop down menu on top-right corner of application.

How to log out from application?

Option to log out is located in drop down menu on top-right corner of application. User can also log out from home screen by pressing ‘back button’ or ‘Home button’ of device.

Do I need internet to use this application?

Internet is required only for ‘Reports’ menu. Services can be used if 2G or 3G is activated in device or device is connected to wi-fi. User can use rest of the application without internet access.

How can I change application's view?

User can change applications view to Tab View or Carousel View by clicling on drop-down menu on top-right corner of application.

How can I login in Application?

User can login by entering registered Email Id and password.

Can I login in my application from any device?

Yes, You can Login in application from any device. Services corresponding to user login will be displayed in application.

Planner

What is ‘Planner’?

Planner allows user to store future events and add reminders in application. User can add meetings, important tasks etc. in application itself. User can set ‘Reminders’ to view scheduled activates in notification bar.

How can I add new Event?

User can add new event by clicking on ‘New Event’ button in Planner menu. Event details can be set as follows,

  • Date: User can enter event date by clicking on the icon.
  • Time: User can enter event time by clicking on the icon
  • Event Name: User can add any event description.
  • Set Alarm: User can check set alarm checkbox to activate notification. If set, application will display event in notification bar on specified date and time.
  • Save: It will save event in application and add small icon against the date of event in the calendar.
  • Cancel: User can cancel the event by clicking on this button.

How can I set Reminder to Event?

User can check ‘set alarm’ checkbox while creating new event to activate notification. If alarm is activated, application will display event in notification bar on specified date and time.

How can I view created Event?

  • Show All Events: User can view all created events by clicking on ‘Show All Events’ in Planner menu. All events will be displayed below calendar.
  • Calendar Small icon will be displayed against the dates for which event is created in the calendar. When user clicks on any date for which event is recorded, event information will be displayed below the calendar.

How can I edit and delete created Event?

  • first,click on show all event and then click on event for edit or delete.
  • Second,when you create event then it will display in below calendar.there you click on event for edit or delete also.

    Premium Calculator

    What is Premium calculator?

    ‘Premium Calculator’ calculates premium to be paid for specific product, depending upon user input.its online calculator. User can click on ‘Calculate’ button to view premium details. ‘Go To Premium Calculator’ will take user to home screen.

    How can I send premium output via SMS?

    Go to Premium Calculator>Select any product>Input values>Click on Calculate>Click on send SMS
    User can send premium output via SMS by clicking on ’Send SMS’ button on premium calculator output screen. Application will open default SMS application or list of applications capable of sending SMS. User can send SMS by entering desired contact number.

    How can I send premium output via E-mail?

    Go to Premium Calculator>Select any product>Input values>Click on Calculate>Click on send Email
    User can send premium output via Email by clicking on ’Send Email’ button on premium calculator output screen. Application will open default Email application or list of applications capable of sending Email. User can send Email by entering desired Email ID.

    Reports

    What is ‘Reports’?

    User can view policy information by selecting any option in ‘Reports’ menu. User can retrieve policy information by providing appropriate input values.

    What are available services in Reports?

  • Policy List (by default it shows 01-April-2013 to current date of financial year 2013-14) (when you hold on list it will show "Policy Detail" and "Customer Detail" service,if you click on "Policy Detail" that it will show policy detail of that policy. and if You click on "Customer Detail" it will show the detail of that customer.)
  • Renewal (if you hold on any list it will pop up menu with 'Customer Detail'. if you click on it it will show customer detail.)
  • Surrender (if you hold on any list it will pop up menu with 'Customer Detail'. if you click on it it will show customer detail.)
  • Maturity (if you hold on any list it will pop up menu with 'Customer Detail'. if you click on it it will show customer detail.)
  • Policy Detail (it will show data with two buttons.one for more detail,in which it will show all more policy detail. and second is view customer detail,in which it will show customer detail.)
  • Commission (it will show commisson of that perticular agent.)
